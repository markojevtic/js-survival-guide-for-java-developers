###### arrays
* arrays are not supported in first version( index is property )
```javascript
let array = [ 15, 5, 123 ];
console.log(array.sort()); 
``` 
* a) [5, 15, 123]
* b) [123, 15, 5]

```javascript
let array = [ 15, 5, 123 ];
console.log(typeof array); 
``` 
* a) 'array'
* b) 'object'
